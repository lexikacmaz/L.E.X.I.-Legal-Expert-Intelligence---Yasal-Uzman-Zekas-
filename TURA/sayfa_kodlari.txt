(TBK’nın 315 veya 362.) maddelerinde belirtilen (on, otuz veya altmış gün) belli bir süre içinde ödemesi, itiraz süresi içinde (İİK m. 62'ye göre) ödeme emrine itiraz etmez ve ödeme süresi içinde borcu (<mark data-markjs="true"><mark data-markjs="true">kirayı</mark></mark>) ödemez ise <mark data-markjs="true"><mark data-markjs="true">kiraya</mark></mark> verenin (alacaklının) kesinleşen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> alacağı için haciz isteyebileceği (İİK m.78 vd) ve icra mahkemesinden (<mark data-markjs="true"><mark data-markjs="true">kiracının</mark></mark>) <mark data-markjs="true"><mark data-markjs="true">kiralanan</mark></mark> taşınmazdan tahliyesini isteyebileceği ihtar edilir. <br>14.  Borçlu-<mark data-markjs="true"><mark data-markjs="true">kiracı</mark></mark> ödeme emrine itiraz etmek isterse ödeme emrinin tebliğinden itibaren (İİK’nın 269/son fıkrasında yazılı istisna hariç) yedi gün içinde itiraz sebeplerini icra dairesine bildirmelidir. İtiraz genel haciz yoluyla takip hükümlerine göre (m. 269/2, 62) yapılır. Ödeme emrine itiraz sebepleri <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine itiraz veya <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesi dışındaki itirazlar şeklinde ikiye ayrılır. <br>15.  İİK’nın 269/b-1 maddesinde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine itiraz ve sonuçları düzenlenmiştir. <mark data-markjs="true"><mark data-markjs="true">Kira</mark></mark> sözleşmesine itiraz, borçlunun (alacaklının iddia ettiği) <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinin varlığını inkâr etmesidir, yani kabul etmemesidir (reddetmesidir). Alacaklı sözlü bir <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinin varlığını iddia ediyorsa, borçlunun böyle bir (sözlü) <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesini inkâr etmesi (kabul etmemesi), <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine itirazdır. Alacaklı yazılı bir <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine dayanmakta ise, borçlu <mark data-markjs="true"><mark data-markjs="true">kiracının</mark></mark> bu sözleşmede kendisine izafe olunan imzayı inkâr etmesi (kabul etmemesi) <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine itirazdır (İİK m. 269/b,1). Borçlu, itirazında <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesini inkâr ettiğini, yani <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesini ve varsa buna ait mukavelenamedeki (yazılı <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki) imzasını kabul etmediğini (reddettiğini) açık ve kesin olarak bildirmelidir; aksi hâlde, borçlu <mark data-markjs="true"><mark data-markjs="true">kiracı</mark></mark>, <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesini kabul etmiş sayılır (İİK m. 269/2 , c.2) (Kuru, B: İcra ve İflas Hukuku, El Kitabı, ... 2013, s. 837).  <br>16.  Hukuk Genel Kurulunun 12.09.2012 tarihli ve  2012/6-343 E., 2012/555 K. sayılı kararında da vurgulandığı üzere İİK’nın 269/2. maddesi gereğince borçlu itirazında <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> akdini ve varsa buna ait sözleşmedeki imzasını açık ve kesin olarak reddetmezse, akdi kabul etmiş sayılacağından, bu durumda <mark data-markjs="true"><mark data-markjs="true">kiralayan</mark></mark> icra mahkemesinden itirazın kaldırılmasını ve tahliyeyi isteyebilir. Öyleyse, <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> akdini kabul eden borçlunun artık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelini ödemiş olduğunu ispat etmesi gerekir.  <br>17.  <mark data-markjs="true"><mark data-markjs="true">Kira</mark></mark> akdi dışındaki itirazlar ise İİK’nın 269/c maddesinde düzenlenmiştir. Borçlunun (<mark data-markjs="true"><mark data-markjs="true">kiracının</mark></mark>), <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine itiraz etmeyip (bilâkis <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesini kabul ederek; m. 269/2, c. 2), <mark data-markjs="true"><mark data-markjs="true">kiranın</mark></mark> daha az olduğu, ödendiği, muaccel olmadığı, takas, <mark data-markjs="true"><mark data-markjs="true">kiralananı</mark></mark> tahliye ettim gibi bir sebeple ödeme emrine itiraz etmesine <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesi dışındaki itirazlar denir (Kuru, 838). Borçlunun aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedeline itirazına öğretide <mark data-markjs="true"><mark data-markjs="true">“kira</mark></mark> borcunun rayiçlere uygun olmadığı” itirazı da denilmektedir (Postacıoğlu, İ: İcra Hukuku Esasları, ... 1982, s. 607). Bu itiraz İİK’nın 269/c maddesinin 1. fıkrasında yazılı sair itiraz kavramı içinde bir itirazdır. Takip talebinde alacaklı yazılı <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine dayanmış ve bu sözleşmede belirlenmiş aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> borcu var ise borçlu <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki imzasını açık ve kesin olarak reddetmezse İİK’nın 269. maddesinin 2. fıkrası hükmüne göre akdi kabul etmiş sayılacağından aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> miktarına itiraz edemez, etse dahi itirazı reddedilir. Dolayısı ile <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinde yazılı <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> borcu da kesinleşir. İtiraz edilmeyerek kesinleşen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinde belirli bir <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> artış oranı var ise bu artış oranına göre belirlenen sonraki dönem aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> miktarları da kesinleşmiş sayılır. Ancak <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinde belirli bir artış oranı yok ise örneğin <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> borcunun enflasyon oranına göre arttırılacağı <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinde kararlaştırılmış ise bu hâlde sadece <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinde yazılı aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> miktarı kesinleşir. <br>18.  Borçlunun <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine veya <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesi dışındaki itirazı ile ilamsız tahliye takibi durur (İİK m. 269/3, c. l). Alacaklının itiraz ile duran tahliye takibine devam edebilmesi için itirazın tebliği tarihinden itibaren altı ay içinde icra mahkemesinden itirazın kaldırılmasını ve tahliye istemesi gerekir.<br>19.  Konut ve çatılı işyeri <mark data-markjs="true"><mark data-markjs="true">kirası</mark></mark> söz konusu ise TBK’nın 347. maddesine göre <mark data-markjs="true"><mark data-markjs="true">kiracı</mark></mark> belirli süreli sözleşmelerin süresinin bitiminden itibaren en az on beş gün önce bildirimde bulunmadıkça, sözleşme aynı koşullarla bir yıl için uzatılmış sayılır. <mark data-markjs="true"><mark data-markjs="true">Kiraya</mark></mark> verenin (<mark data-markjs="true"><mark data-markjs="true">kiracı</mark></mark> istemez ise) <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> süresinin bitiminde <mark data-markjs="true"><mark data-markjs="true">kiracıyı</mark></mark> taşınmazdan çıkarma hakkı kural olarak yoktur. <mark data-markjs="true"><mark data-markjs="true">Kiracı</mark></mark> taşınmazı boşaltmak istemez ise <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> süresi aynı şartlarla bir yıl daha uzatılmış sayılır. 01.07.2012 tarihinde yürürlüğe giren TBK’nın 344. maddesinde tarafların yenilenen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> dönemlerinde uygulanacak <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedeline ilişkin anlaşmaları sınırlandırılmış olup, <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedellerinin belirlenmesine ilişkin bir takım esaslara yer verilmiştir. Ancak 6217 sayılı Kanun’un geçici 2. maddesi uyarınca <mark data-markjs="true"><mark data-markjs="true">kiracısı</mark></mark> tacir veya tüzel kişi olan işyeri <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmelerinde TBK’nın 344. maddesinin de içinde bulunduğu dokuz maddenin yürürlüğü 8 yıl süre ile ertelenmiştir. Belirtilen madde hükmüne göre erteleme süresi bitimi olan 01.07.2020 tarihine kadar <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmelerindeki hükümler sözleşme serbestisi gereğince uygulanacaktır. <br> 20.  Somut olayda alacaklının 01.07.2007 başlangıç tarihli <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine dayalı ilamsız tahliye takibinde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki artırım şartından kaynaklanan Aralık 2013 ile Temmuz 2014 tarihleri arasındaki <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> farklarının tahsilini talep ettiği, ödeme emrinin borçluya tebliği üzerine borçlu vekilinin yasal süresinde verdiği itiraz dilekçesinde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki imzasını açık ve kesin olarak reddetmediği, itiraz dilekçesinde aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin 1250TL olduğunu ve <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> borcunu ödediğini bildirmek suretiyle aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> miktarına itiraz ettiği, alacaklının itirazın kaldırılması ve tahliye isteği ile icra mahkemesine başvurduğu, borçlu vekilinin cevap dilekçesinde alacaklı tarafından müvekkiline gönderilen ... 6. Noterliğinin 29.05.2014 tarihli ve 08930 yevmiye nolu <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin artırımına ilişkin ihtarnamenin "ödenmekte olan aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark>" bölümünde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin 1.250TL+ <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> stopajı olarak açık ve net olarak belirtildiğini, bu ihtarname ile <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin aksine bir sözleşme mevcut olmamakla şifahen oluştuğunu ve aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin 1.250TL + stopaj şeklinde yürürlükte bulunduğunun ispatlandığını beyan ettiği görülmektedir. <br>21.  Borçlu itirazında takibe dayanak <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki imzasını açık ve kesin olarak reddetmediğinden İİK’nın 269/2. maddesi gereğince akdi kabul etmiş sayılır. Bu durumda 01.07.2007 başlangıç tarihli <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesi tarafları bağlar. <mark data-markjs="true"><mark data-markjs="true">Kiralanan</mark></mark> işyerinin <mark data-markjs="true"><mark data-markjs="true">kiracısı</mark></mark> tüzel kişi olduğu hâlde 6217 sayılı Kanun’un geçici 2. maddesi uyarınca 01.07.2020 tarihine kadar TBK’nın 344. maddesi uygulanmayacağından ve takip talebinde istenilen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> dönemi erteleme kapsamında kaldığı için sözleşme serbestisi gereğince <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinin 7. maddesinde yazılı “yıllık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> artış oranı %20 olarak arttırılacaktır”şeklindeki artış şartı geçerlidir. O hâlde sözleşmenin icra takibine konu edilen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> döneminde aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> parasının sözleşmedeki artırım şartına bağlı olarak belirlenmesi gerekir. <br>22.  Nitekim Hukuk Genel Kurulunun 10.04.2013 tarihli ve 2012/3-1215 E., 2013/465 K. sayılı kararında sözleşme serbestisi ilkesi çerçevesinde tarafların özgür iradeleri ile kararlaştırdıkları <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki artışa ilişkin  hükmün uygulanacağı, ödenmesi gereken <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> miktarının belirlenmesinde sözleşme hükmünün esas alınması gerektiği, Hukuk Genel Kurulunun 03.04.2013 tarihli ve 2012/6-1401 E., 2013/425 K. sayılı kararında da borçlunun karşı çıkmadığı <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin belirlenmesinin ölçütleri ve artış oranlarının özel şartlar bölümünde açıkça düzenlendiği, dar yetkili icra mahkemesinin itiraz ederken sebeplerini ayrıntısıyla bildiren borçlunun itirazlarının kaldırılıp kaldırılmayacağına karar verirken, tarafların kabulünde olan  sözleşme hükümlerini değerlendirmek ve ayrı bir yargılama da gerektirmediğinden sözleşme üzerinden tarafların sav ve savunmalarını incelemek durumunda olduğu benimsenmiştir.<br>23.  Diğer taraftan alacaklının <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> tespit davası açmak üzere borçluya gönderdiği ... 6. Noterliğinin 29.05.2014 tarihli ve 08930 yevmiye nolu <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin artırımına ilişkin ihtarnamenin "ödenmekte olan aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark>" bölümünde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedeli 1.250TL+ <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> stopajı olarak gösterilmiş olup, “ödenmekte olan” tabiri aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin 1.250TL olduğunun kabul edildiği anlamına gelmez. Bu nedenle söz konusu ihtarname aylık <mark data-markjs="true"><mark data-markjs="true">kiraya</mark></mark> itirazın ispatı için İİK’nın 269/c maddesinde yazılı olan belgeler niteliğinde değildir. Ayrıca ihtarnamede takip talebinde istenen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> farklarına ilişkin <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> döneminden sonraki <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> dönemi olan 01.07.2014 tarihinden itibaren aylık <mark data-markjs="true"><mark data-markjs="true">kiranın</mark></mark> 10.000TL+<mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> stopajı olarak ödenmesi, aksi takdirde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin artırılması için dava açılacağının ihtar edilmesinden aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin 1250TL olduğu anlamı da çıkarılamaz.<br>24.  Takip talebinde 01.07.2007 tarihli 12 ay süreli yazılı <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesine dayalı olarak 05.12.2013- 05.07.2014 tarihleri arasında 8 aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> farkı toplamı 55.691,36TL asıl alacak talep edildiği, <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesinde aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> miktarı 2.750TL olup, sözleşmenin özel şartlar kısmının 7. maddesinde “yıllık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> artış oranı %20 olarak artırılacaktır” hükmü bulunduğu, borçlunun yazılı <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki imzaya itiraz etmediğinden <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> aktinin dolayısı ile aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> miktarı ile <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> artış oranının kesinleştiği, 01.07.2013 <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> dönemi için artış oranına göre kesinleşen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> bedelinin 8.211,42TL olduğu, borçlunun takipten önce her ay için 1250TL miktarındaki <mark data-markjs="true"><mark data-markjs="true">kirayı</mark></mark> alacaklının hesabına yatırdığı, ödenmeyen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> farkının 6.961TL olduğu, takip talebinde 8 aylık <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> farkı tutarı olan 55.691,36TL’nin takip talebinde talep edildiği, borçlunun örnek 13 ödeme emrine karşı 7 gün içinde icra dairesine <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> akdi dışında aylık <mark data-markjs="true"><mark data-markjs="true">kiranın</mark></mark> 1.250TL olduğu ve bu miktarda ödeme yaptığı için borcu olmadığı itirazında bulunduğu, 30 günlük ödeme süresi içinde icra dairesine bir ödeme yapmadığı, İİK’nın 269/c maddesindeki belgelerle itirazını ispatlayamadığı anlaşılmaktadır.  <br>25.  O hâlde <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> sözleşmesindeki artırım oranına uygun <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> borcunu ödemeyen borçlu temerrüde düştüğünden mahkemece itirazın kaldırılması ve tahliyeye karar verilmesi gerekir. <br>26.  Hukuk Genel Kurulunda yapılan görüşmeler sırasında borçlu tarafından İİK’nın 269/c maddesinde düzenlenen <mark data-markjs="true"><mark data-markjs="true">kira</mark></mark> borcunun sair bir sebeple istenemeyeceği hükmüne göre ... 6. 	<script>
        (function ($) {
            $.fn.countTo = function (options) {
                options = options || {};

                return $(this).each(function () {
                    // set options for current element
                    var settings = $.extend({}, $.fn.countTo.defaults, {
                        from: $(this).data('from'),
                        to: $(this).data('to'),
                        speed: $(this).data('speed'),
                        refreshInterval: $(this).data('refresh-interval'),
                        decimals: $(this).data('decimals')
                    }, options);

                    // how many times to update the value, and how much to increment the value on each update
                    var loops = Math.ceil(settings.speed / settings.refreshInterval),
                        increment = (settings.to - settings.from) / loops;

                    // references & variables that will change with each update
                    var self = this,
                        $self = $(this),
                        loopCount = 0,
                        value = settings.from,
                        data = $self.data('countTo') || {};

                    $self.data('countTo', data);

                    // if an existing interval can be found, clear it first
                    if (data.interval) {
                        clearInterval(data.interval);
                    }
                    data.interval = setInterval(updateTimer, settings.refreshInterval);

                    // initialize the element with the starting value
                    render(value);

                    function updateTimer() {
                        value += increment;
                        loopCount++;

                        render(value);

                        if (typeof (settings.onUpdate) == 'function') {
                            settings.onUpdate.call(self, value);
                        }

                        if (loopCount >= loops) {
                            // remove the interval
                            $self.removeData('countTo');
                            clearInterval(data.interval);
                            value = settings.to;

                            if (typeof (settings.onComplete) == 'function') {
                                settings.onComplete.call(self, value);
                            }
                        }
                    }

                    function render(value) {
                        var formattedValue = settings.formatter.call(self, value, settings);
                        $self.html(formattedValue);
                    }
                });
            };

            $.fn.countTo.defaults = {
                from: 0,               // the number the element should start at
                to: 0,                 // the number the element should end at
                speed: 1000,           // how long it should take to count between the target numbers
                refreshInterval: 100,  // how often the element should be updated
                decimals: 0,           // the number of decimal places to show
                formatter: formatter,  // handler for formatting the value before rendering
                onUpdate: null,        // callback method for every time the element is updated
                onComplete: null       // callback method for when the element finishes updating
            };

            function formatter(value, settings) {
                return value.toFixed(settings.decimals);
            }
        }(jQuery));
